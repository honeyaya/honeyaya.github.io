c++ 两个方面体现了重用

  • 面向对象的设计思想，继承，多态以及标准类库

  • 泛型程序设计，包括模板机制以及标准模板库STL 就是将一些常用的数据结构写成模板

    ​

c++ STL 大致由 容器，迭代器，算法，函数对象 四个部分组成

* 容器是可容纳各种数据类型的数据结构

      1.包括**顺序容器**和**关联容器**

      2.还有一类不提供真正的用于存储元素的数据结构实现，称作容器适配器

      3.容器适配器不支持迭代器，由程序员选择合适的底层数据结构

* 迭代器是用于依次存取容器中元素的指示器

      1.实质上是一类泛化了的指针

      2.通过使用迭代器，开发者可以操作数据结构而无需关心其内部实现

* STL还提供了一些常见算法的函数模板，使得算法与数据结构分离

      1.可以对自定义的数据结构使用这些算法

* 函数对象即重载了操作符()的对象

      1.类的实例都可以称作为函数对象
