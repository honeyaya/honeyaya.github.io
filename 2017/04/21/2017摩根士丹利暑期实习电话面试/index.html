2017摩根士丹利暑期实习电话面试

一通来自上海的电话，莫名的响了很久

难道是？

回拨，占线

15min之后，“我是你摩根士丹利的第一轮面试官，下午6：30有时间吗？英语面试”

“由于挂了三次的电话，弱弱的“好哒好哒””

准点电话：面试流程“简单的自我介绍+ 专业基础问题+面试者提问”

真心英语需要提高

这里简单的记录下中间的专业问题：

  1. 指针是什么？

  2. 引用是什么？

  3. ArrayList是什么？

  4. stackflow是什么？

  5. uml是什么？

  6. abstrac干嘛用的？

  7. sql rejection是什么？

  8. data bus and control bus ？

  9. Dos attack？

  10. 北京的地铁怎么表示？

  11. process && thread区别？

  12. 桌子有1元，5角，1角混合在一起，有一台电子秤，怎么能知道how much？

    1. 当时就在想可以获取一个的重量，x w1 + y w5 + z * w11 = total weight （x,y,z为三种货币的个数，w1,w5,w11分别是对应一枚的重量）然后妥妥卡死在这里了
    2. 面试官有讲知道大概的钱就可以了，英语不好，这句直接忽略了，然后在求准确的钱的道路上越走越远。。。

    其实想一下也就是了，等比例的抽取，就可以获取了

整体面的不好，主要是没办法用英语表达自己的想法，中间无耻的问面试官“can I speak chinese?” “No” 面试官就那么拒绝了

幸亏专业英语，我大概知道面试官说的是什么，嗯，估计做分母了

好好提高英语口语吧，其实问题都很简单，但是一下子进入英语环境以及本身电话面试就很紧张，很多点发挥不出来
