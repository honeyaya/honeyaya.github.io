正则表达式匹配

思路分析：每次从字符串中拿出一个字符和模式中的字符匹配，如果模式中的字符ch是. 那么可以完成匹配，如果ch不是.并且字符串中的字符也是ch,也是相互匹配的，当两者匹配时，接着匹配后面的字符。

当模式中的第二个字符不是*，如果字符串中的第一个字符和模式中的第一个字符匹配，那么字符串和模式均向后移动一个字符，然后继续匹配，如果字符串中的第一个字符和模式中的第一个字符不相匹配，则直接返回false

当模式中的第二个字符是 的时候，此时可能有多种匹配方式，一种是直接在模式上向后移动两个字符，相当与 和前面的字符被忽略， * 可以匹配0个字符。如果模式中的第一个字符和字符串中的第一个字符相匹配，则在字符串向后移动一个字符，模式串有两个选择： 可以向后移动两个，也可以保持不变。

void InOrder(TreeNode *root){
  if(root == NULL) return ;
  TreeNode *p = root;
  stack<TreeNode> st;

  while(p!=NULL || !st.empty()){
      while(p!=NULL) {
          st.push(p);
          p = p -> left;
      }

      if(!st.empty()){
          p = st.top();
          cout << p -> val << endl;
          st.pop();
          st.push(p->right); 
      }
  }
}

百望山

14/04/17的北京天气不错，就打算去百望山

简单的查了下攻略，说是百望山是因为佘老太君曾望子归来的地方，不过看到说氧负离子很多，想想可以呼吸一口，沁人心脾的那种醉~

9点出的门，也就两站，近的简直了

好久不坐地铁，表示不太习惯

从小路上山，感觉有点老了，15年爬香山的时候，还很壮年呢

c++ STL

STL

String

string 类是一个模板类，用意字符串操作

1.构造赋值

2.字符访问

3.容量

4.搜索

5.比较

6.修改器

1.+ or  += 连接
2.append(string,int,int)  s2.append(s2,5,6)  5是位置，6是长度
3.insert()
4.push_back()/pop_back()
5.replace(int,int,string)  起始位置以及长度改成某字符串

7.流输入输出

8.类型转换

c++ 两个方面体现了重用

  • 面向对象的设计思想，继承，多态以及标准类库

  • 泛型程序设计，包括模板机制以及标准模板库STL 就是将一些常用的数据结构写成模板

    ​

c++ STL 大致由 容器，迭代器，算法，函数对象 四个部分组成

* 容器是可容纳各种数据类型的数据结构
    1.包括**顺序容器**和**关联容器**
    2.还有一类不提供真正的用于存储元素的数据结构实现，称作容器适配器
    3.容器适配器不支持迭代器，由程序员选择合适的底层数据结构                  

* 迭代器是用于依次存取容器中元素的指示器
    1.实质上是一类泛化了的指针
    2.通过使用迭代器，开发者可以操作数据结构而无需关心其内部实现      

* STL还提供了一些常见算法的函数模板，使得算法与数据结构分离
    1.可以对自定义的数据结构使用这些算法

* 函数对象即重载了操作符()的对象
    1.类的实例都可以称作为函数对象
